Notes

  • Module Bundler: Webpack 4
  • Languages used: HTML, SCSS / CSS, JavaScript, jQuery, PHP
  • I have coded the save functionality in the save_calc.php which then rediects back to the index page. This could also have been done within the index page if required.
  • To view and use the calculator, you can just drag the index.html into the browser from the dist folder
  • For saving and viewing the saved sums, you will need to run it on a PHP server
